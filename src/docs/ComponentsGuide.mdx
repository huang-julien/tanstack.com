# Components Guide

Higher-level components built on top of UI primitives. These implement specific patterns and interactions used throughout tanstack.com.

---

## Avatar

Displays a user avatar with automatic fallback chain: profile image → OAuth image → initials → generic icon.

<Canvas of={AvatarStories.WithImage} />

### Fallback priority

1. `image` — Direct profile image URL
2. `oauthImage` — OAuth provider avatar
3. `name` / `email` — Generates initials (e.g. "TL" for "Tanner Linsley")
4. Generic user icon

### Sizes

Six sizes from `2xs` (16px) to `xl` (80px):

<Canvas of={AvatarStories.Sizes} />

## Collapsible

An expandable section supporting both controlled and uncontrolled modes. Includes `CollapsibleTrigger` and `CollapsibleContent` sub-components.

<Canvas of={CollapsibleStories.Default} />

### Modes

| Mode             | Usage                                                          |
| ---------------- | -------------------------------------------------------------- |
| **Uncontrolled** | Pass `defaultOpen` for initial state, component manages itself |
| **Controlled**   | Pass `open` and `onOpenChange` for external state control      |
| **Render props** | Pass a function as children to access `{ open, toggle }`       |

## PaginationControls

Full pagination bar with page navigation, page size selector, and item count display.

<Canvas of={PaginationStories.Default} />

### Features

- Previous/Next buttons with disabled states
- Page size selector with configurable options (default: 10, 25, 50, 100)
- Item count display with optional filtered count
- Sticky mode for fixed positioning
- Custom item label (e.g. "users", "entries")
