The internal component library powering [tanstack.com](https://tanstack.com). Built with **React**, **Tailwind CSS v4**, **Radix UI**, and **TanStack Router**.

---

## Getting Started

Browse the sidebar to explore available components. Each one has:

- **Stories** — interactive examples showing the component in different states
- **Docs** — auto-generated API reference from prop types and JSDoc comments

## Architecture

### UI Primitives (`src/ui/`)

Low-level building blocks used across the site:

| Component      | Description                                         |
| -------------- | --------------------------------------------------- |
| **Button**     | Polymorphic button with variants, colors, and sizes |
| **Badge**      | Status pill with 8 color variants                   |
| **FormInput**  | Styled input with configurable focus ring color     |
| **InlineCode** | Inline `code` styling for text                      |

### Components (`src/components/`)

Higher-level components that compose UI primitives:

| Component              | Description                                                 |
| ---------------------- | ----------------------------------------------------------- |
| **Avatar**             | User avatar with image, initials, or fallback icon          |
| **Card**               | Polymorphic container with shadow and border styling        |
| **Collapsible**        | Expandable section with controlled/uncontrolled modes       |
| **Dropdown**           | Radix-based dropdown menu                                   |
| **Select**             | Custom select dropdown with optional logo icons             |
| **Tooltip**            | Floating UI tooltip with placement options                  |
| **Breadcrumbs**        | Section breadcrumb with optional table-of-contents dropdown |
| **ButtonGroup**        | Grouped button container with dividers                      |
| **Spinner**            | Animated loading indicator                                  |
| **FeatureGrid**        | Checklist grid for feature callouts                         |
| **PaginationControls** | Full pagination bar with page size selector                 |
| **TrustedByMarquee**   | Auto-scrolling brand logo marquee                           |

### Route-Dependent Components

Components that require TanStack Router context:

| Component            | Description                                          |
| -------------------- | ---------------------------------------------------- |
| **Navbar**           | Site-wide navigation bar with menu, search, and auth |
| **Footer**           | Site footer with link grid                           |
| **BottomCTA**        | Call-to-action banner with router link               |
| **ShowcaseCard**     | Showcase project card with voting                    |
| **FrameworkCard**    | Framework selector card with copy-to-clipboard       |
| **Logo / LogoColor** | TanStack brand logos with context menu               |

## Tech Stack

- **TanStack Start** — Full-stack React framework
- **TanStack Router** — Type-safe routing
- **Tailwind CSS v4** — Utility-first CSS
- **Radix UI** — Accessible primitives (Dropdown, Dialog, Toast)
- **Floating UI** — Tooltip positioning
- **lucide-react** — Icon library
